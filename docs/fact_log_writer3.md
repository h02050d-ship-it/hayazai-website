## 表現を弱めた・見送った論点まとめ

1. サネにボンドNG → 1ソースのため「〜とされています」+理由説明
2. ビス長2〜2.5倍 → 1ソースのため「目安として紹介されています」
3. 根太間隔303mm/450mm → 割れているため条件付き両論併記
4. LL45の「遮音マット11mm+合板12mm+無垢15mm=38mm」 → 裏取り不可。具体11mmは書かず「40mm前後の構成例」に
5. 二重床ならLL45不要論 → 裏取り不可のため見送り
6. 「6畳上張り約10時間」 → 実例（10数畳11時間等）の紹介に変更
7. 床暖対応/非対応の同室併用NG → 出所（専門店の明言)を示す形
8. 最後の一列の「30度落とし」 → 実践者のコツとして紹介
9. 畳厚 → 50〜60mmと幅があるため「必ず実測」を併記
