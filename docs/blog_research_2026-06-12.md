## 領域1: 購入前の疑問・不安

### 1-1. 無垢フローリングのデメリット・後悔ポイント
- **読者の悩み**: 「無垢にして後悔しないか」「営業に勧められたが本当に大丈夫か」。検索上位は「後悔」「デメリット」系記事が圧倒的。
- **押さえるべき事実**:
  - 後悔の定番5つ: ①傷・凹みがつきやすい ②水・食べこぼしのシミ ③隙間・反り・突き上げ（膨張収縮） ④価格が高い ⑤定期メンテナンスが必要。
  - 価格相場: 無垢は1万円/㎡程度〜、複合は5,000円/㎡程度〜。ただし「挽板複合は無垢並みに高い」「安い無垢（杉・パイン）は突板複合より安い」ので「無垢=高い」は単純化しすぎ。
  - ホットカーペット・こたつは床暖対応品でもNG（局所高温・湿気こもりで割れ・反り）。断熱シートを敷けば条件付きで可。
  - 無垢材は「傷はつけども目立ちづらい」— 経年で色が深まり艶が出ると傷が馴染む（針葉樹特有の訴求点）。
- **よくある誤解**: 「無垢=メンテが大変」（ウレタン塗装無垢ならほぼメンテフリー）、「無垢=必ず高い」、「傷=劣化」（複合は表層剥離で寿命だが無垢は削り直せる）。
- **情報源**: https://www.muku-flooring.jp/blog/2019/05/07/19510/ / https://www.atopico.com/mukuzai/blog/1094.html / https://www.daiken.jp/buildingmaterials/flooring/trinity/columnipe/006/ / https://isetsu.co.jp/blogcat/blog/solid-flooring-regret/ / https://www.2mfactory.jp/blogs/mmnote/solid-flooring

### 1-2. 無垢 vs 複合フローリング比較
- **読者の悩み**: 「結局どっちがいいのか」「複合の挽板・突板・シートの違いがわからない」。
- **押さえるべき事実**:
  - 複合は3種: 挽板（2mm厚無垢表層、6,000円/㎡〜）、突板（0.3mm程度、4,000円/㎡〜）、シート（印刷、最安）。
  - 無垢は調湿・断熱（足が冷たくない）・経年美・削り直し可。複合は寸法安定・床暖/防音対応の選択肢が広い・施工が楽。
  - 複合は表層が剥がれたら終わり（張替え）、無垢はサンディング再生可＝長期では無垢が経済的という論点。
- **よくある誤解**: 「複合=安物」（挽板は高級品）、「無垢は床暖不可」（対応品あり）。
- **情報源**: https://www.cas-online.jp/blog/pickup/12074/ / https://kashida-m.co.jp/column/202405_03/ / https://www.muku-flooring.jp/blog/2025/10/16/56076/ / https://www.daiken.jp/buildingmaterials/flooring/trinity/columnipe/002/

### 1-3. 節あり/節なし（グレード）の選び方
- **読者の悩み**: 「節有・上小節・無節って何が違う？」「節ありは安いけど大丈夫？」
- **押さえるべき事実**:
  - グレードは見た目と価格の違いであり、強度・香り・調湿性・抗菌性は変わらない。
  - 無節は丸太の中心付近からしか取れず希少→高価。節あり=ラフで力強い、上小節=中間、無節=上品。
  - 針葉樹は枝が細く多い→節が小さく多い。広葉樹は太く少ない。死に節・埋木処理の有無も品質ポイント。
- **よくある誤解**: 「節あり=低品質・弱い」。
- **情報源**: https://www.muku-flooring.jp/blog/2020/05/15/23334/ / https://www.cas-online.jp/blog/feature/8924/ / https://www.yoshidaseizai.co.jp/column/about_rank.html / https://hayazai.com/blog/how-to-choose-grade.html（既存記事・検索上位に表示確認済み）

### 1-4. UNI / OPC（一枚物）/ 乱尺の違い
- **読者の悩み**: 「商品名のUNIって何？」「一枚物は高いけど価値ある？」
- **押さえるべき事実**:
  - OPC=One Piece、継ぎ目のない一枚物。木目が通り高級感・奥行き感。広葉樹では製造量の10%以下で希少・割高。
  - UNI=短材を縦にフィンガージョイントした規格（乱尺を施工しやすく工場で繋いだもの）。価格と施工性のバランス◎。
  - 乱尺=400〜1200mm程度の不揃い長さがランダム梱包。4方本実。安いが割付の手間。
- **よくある誤解**: 「UNI=集成材で偽物」（横方向には貼り合わせていない、無垢材である）。
- **情報源**: https://www.muku-flooring.jp/blog/2019/06/14/20103/ / https://www.zaimoku-ya.com/guide/qa/flooringkeijo/ / https://www.e-kenzai.jp/info/column/20240725_kikaku / https://www.diy-shop.jp/second/muku-flooring/type-grade.html

### 1-5. 厚み15mm / 30mmの違い
- **読者の悩み**: 「厚い方がいい？」「30mmは何が違う？」
- **押さえるべき事実**:
  - 標準は15mm。30mm厚は捨て貼り合板なしで根太に直接張れる（根太レス・構造強度を床材自身で確保）。
  - 厚いほど高価で重い。30mmは杉・桧などの軽い針葉樹が中心（コスパと重量の関係）。
  - 厚みがあるほど断熱・蓄熱感が増し「踏み心地」が変わる、というユーザー評価が多い。
- **よくある誤解**: 「薄い=安物」（硬い広葉樹なら15mmで十分）。
- **情報源**: https://daityu.shop/column/thickness.html / https://kashida-m.co.jp/news/202410_02/ / https://www.archi-c.com/床板は厚い方がいいか/ / https://morinogakko.jp/diy-washitsu/

### 1-6. 塗装品/無塗装、オイル/ウレタンの選び方（購入時）
- **読者の悩み**: 「無塗装とオイル塗装どっちを買うべき？」「ウレタンだと木の良さが消える？」
- **押さえるべき事実**:
  - 無塗装: 木そのものの足触り・調湿・香り最大。シミ・手垢黒ずみがつきやすい。
  - オイル: 浸透型・塗膜なし。質感を保ちつつ多少の保護。年1〜2回の再塗装。凹みのアイロン補修・ペーパー研磨が可能。
  - ウレタン: 塗膜型。水・汚れに強くメンテほぼ不要、塗膜寿命20〜30年。ただし調湿停止・足触りはプラスチック的・部分補修不可。
  - 桧は油分が多く着色塗装には不向き（クリア系推奨）という固有論点あり。
- **よくある誤解**: 「オイル塗装すれば水を完全に弾く」（シミは普通にできる）。
- **情報源**: https://isetsu.co.jp/blogcat/blog/urethane_oil_difference/ / https://crazy-wood.jp/archives/60 / https://mukurich.jp/muku-flooring-maintenance-oil-urethane/ / https://www.muku-flooring.jp/tosou/

### 1-7. 樹種比較（桧・杉・パイン・オーク等）
- **読者の悩み**: 「桧と杉どっちにする？」「オークと比べて桧は柔らかすぎない？」
- **押さえるべき事実**:
  - 硬さレベル: 杉1 ＜ 桧・パイン2 ＜ オーク4（傷つきにくさはオーク優位、温かさ・足触りは針葉樹優位）。
  - 価格目安: 杉3,000〜7,000円/㎡、桧5,000〜8,000円/㎡、オークはそれ以上。
  - 桧の固有メリット: 香り（α-ピネン等のリラックス効果）、耐水・耐久性（寺社建築・法隆寺）、抗菌成分（国産桧はヒノキチオールでなくα-カジノール等のセスキテルペン ※正確に書くと差別化になる）、白い木肌→飴色への経年変化。
  - 杉vs桧: 杉は柔らかく温かい（寝室・子供部屋向き）、桧はやや硬く水回りや人が集まる場所向き。
- **よくある誤解**: 「ヒノキチオール=国産桧の成分」（実はタイワンヒノキ・ヒバに多く、国産桧にはほぼ含まれない）。
- **情報源**: https://www.e-kenzai.jp/info/column/20240926_katasahikaku / https://www.cas-online.jp/blog/pickup/9512/ / https://hameitaya.com/owner-blog/column/20192/ / https://www.karumoa.co.jp/suvaliteair/column/hinoki/ / https://www.atopico.com/blog/23995.html

### 1-8. 床暖房との相性
- **読者の悩み**: 「無垢で床暖は無理？」「対応品って何が違う？」
- **押さえるべき事実**:
  - 床暖対応無垢は乾燥処理を強化した専用品。樹種により対応温度が違う（50℃〜80℃）。温水式の方が電気式より木に優しい。
  - 対応品と一般品を同じ部屋で併用不可（収縮率が違い反り・割れの原因）。
  - 暖房期は隙間が出るが夏に戻る、と事前説明するのが業界標準。
- **情報源**: https://www.muku-flooring.jp/wood_list_yukadan.html / https://www.logrenove.jp/basic/16915/ / https://www.zaimoku-ya.com/guide/yukadan/ / https://www.cas-online.jp/blog/pickup/12649/

### 1-9. ペット・子供との相性
- **読者の悩み**: 「犬が滑って股関節を痛めないか」「子供が傷だらけにしないか」「おしっこのシミ」。
- **押さえるべき事実**:
  - 無垢は爪が立つためコーティング複合より滑りにくい＝ペットの関節保護に有利、という逆転の発想が訴求点。
  - 浮造り仕上げは滑り防止+傷が目立ちにくい。柔らかい針葉樹はクッション性があり転倒時の衝撃緩和。
  - 無垢+赤ちゃん: 接着剤不使用・ホルムアルデヒドほぼゼロ・調湿でダニカビ抑制・転んでも痛くない。
  - ペットの粗相は早期拭き取り＋撥水系オイル/蜜蝋で予防。
- **よくある誤解**: 「滑り抵抗値が高い床=ペットに安全」（爪のかかり方の方が重要との専門店見解）。
- **情報源**: https://hags-ec.com/column/muku-flooring-pets/ / https://www.muku-flooring.jp/blog/2022/10/10/51348/ / https://isetsu.co.jp/blogcat/blog/24-10-27/ / https://www.e-kenzai.jp/info/column/20220706_kosodate / https://crazy-wood.jp/archives/17

### 1-10. 賃貸での無垢フローリング
- **読者の悩み**: 「賃貸でも無垢の床にしたい」「原状回復できる？」
- **押さえるべき事実**:
  - 釘・ボンド不要の「敷くだけ/置くだけ」「はめ込みロック式」無垢フローリングが存在（toolbox無垢床タイル、イージーロック等）。
  - 退去時は撤去するだけ。接着剤不使用でシックハウスの心配なし。
  - 既存床に固定する通常施工は原状回復不可なのでNG。
- **情報源**: https://99diy.tokyo/tintai_flooring/ / https://www.r-toolbox.jp/stories/productstory/1871/ / https://www.diy-shop.jp/floor-tile/choose-lay.html

### 1-11. 産地ブランド（東濃桧・尾鷲桧・天竜桧など）
- **読者の悩み**: 「産地で何が違う？」「天竜桧って良いの？」
- **押さえるべき事実**:

### 1-12. 価格相場と「安く買う方法」
- **読者の悩み**: 「桧フローリングの適正価格は？」「どこで買うと安い？」
- **押さえるべき事実**:
  - 紀州桧 節あり無塗装で8,400〜12,500円/㎡程度、ウレタン塗装品は14,000〜20,000円/㎡の例。1ケース=半坪（約1.66㎡）入りが標準（90mm幅10枚等）。

  - サンプル請求→照明で色が変わる（昼の自然光は白く、夜の電球色はオレンジに）、小サンプルと全面では印象が激変、天然材ゆえ1枚ごとの色ばらつきは必然、という「サンプルの正しい見方」。

## 領域2: DIY施工

### 2-1. 工法の基礎（捨て貼り/根太張り/直貼り）
- **読者の悩み**: 「捨て貼りって必要？」「根太に直接張っちゃダメ？」
- **押さえるべき事実**:
  - 捨て貼り工法: 根太+12mm構造用合板+フローリング。防音・安定性・きしみ減。DIYには捨て貼り推奨が定説。
  - 根太張り工法: 根太に直接。材料費・工期減だがたわみやすい。15mm無垢を根太張りする場合は根太間隔303mm推奨。
  - 30mm厚なら捨て貼り省略で根太直張り可（杉30mmで6畳6時間のDIY事例あり）。
  - コンクリート直貼り（マンション）: 接着剤のみで張る工法。無垢単体では遮音が取れずほぼ専用品/マット併用になる。
- **情報源**: https://trs.jp.toto.com/blog/68/post_3143w-dispose_pasting-and-joist_pasting-method / https://www.cas-online.jp/blog/feature/10310/ / https://www.hiro-secondwork.com/entry-flooring-construction-method/ / https://www.diy-shop.jp/second/flooring/constr-method.html

### 2-2. 既存床への重ね張り（上張り）
- **読者の悩み**: 「古い床を剥がさず張れる？」「ドアが開かなくなるって本当？」
- **押さえるべき事実**:
  - 床高アップ（12〜15mm）でドア・建具との干渉が最頻トラブル。開き戸の干渉チェックが必須。
  - 既存床のきしみ・たわみは上張りでは直らない（剥がして下地からやり直し）。
  - 巾木は先に外す。既存床のワックスは完全除去（接着不良の原因）。ボンドは必ず油性ウレタン系（木工用水性は床鳴り・反りの元）。
  - DIYで一番チャレンジしやすいのは上張り、最難関は剥がしての張替え。
- **情報源**: https://sumosta.jp/archives/446 / https://www.heartcoat.jp/blog/column/1900/ / https://www.cas-online.jp/blog/feature/3433/ / https://www.diy-shop.jp/second/flooring/mansion-overlay.html

### 2-3. 必要な道具と釘・ビス・ボンドの使い分け
- **読者の悩み**: 「何を買い揃えればいい？」「釘とビスどっちで留める？」
- **押さえるべき事実**:
  - 固定3方式: ①フロア釘+ポンチ+金槌（昔ながら・安価・体力勝負）②フローリング用スリムビス+インパクト（DIY向き・実が割れにくい細軸を選ぶ）③フィニッシュネイラ+コンプレッサ（早いが投資大）。
  - 長さの目安: 釘は12mm厚で38mm、ビスは床材厚の2〜2.5倍（15mm厚なら30mm以上）。
  - ボンドは必ずウレタン系（根太ボンド）。水性ボンドは反り・床鳴りの原因。**実（サネ）にはボンドを付けない**（伸縮の逃げ場が消える）。
  - 基本手順: 根太ボンド→当て木でサネを噛ませる→雄ザネに斜め45度で釘/ビス。
  - その他道具: 丸ノコ、スケール、さしがね、当て木、ゴムハンマー、スペーサー、ポンチ。
- **情報源**: https://www.muku-flooring.jp/blog/2021/04/10/44313/ / https://99diy.tokyo/wooden_flooring/ / https://nohmiso.com/woodenfloor-howto-diy/ / https://www.diy-shop.jp/info/diy_mf.html

### 2-4. 割付のコツと壁際クリアランス
- **読者の悩み**: 「どこから張り始める？」「最後の列が入らない」。
- **押さえるべき事実**:
  - 張り始めは基準線を出して壁から。継ぎ目は2尺ずらし or 乱張りで一直線にしない。必ず仮並べ（仮置き）して色・木目バランスを調整。
  - 板同士のクリアランス0.3mm（名刺1枚程度）、壁際は5〜10mmの伸縮スペース必須（巾木で隠れる）。
  - スペーサーは2尺（606mm）ピッチ、抜き忘れ防止に目立つ色を使う。
  - 最後の一列: 幅を測って雄ザネ側をカット、側面下部を30度に落とすとはめやすい。ゴムハンマーで上から叩いてはめる。列同士を先に繋いでから斜めに倒し込む。
- **情報源**: https://www.muku-flooring.jp/blog/2022/02/15/50815/ / https://www.muku-flooring.jp/manual/ / https://99diy.tokyo/wooden_flooring/ / https://www.yukazaiya.com/hpgen/HPB/entries/20.html

### 2-5. 巾木・見切り材・敷居との取り合い
- **読者の悩み**: 「壁際の隙間はどう隠す？」「廊下と部屋の境目はどうする？」
- **押さえるべき事実**:
  - 巾木は出巾木/入巾木/同面巾木の3種。DIYでは後付けの「付け巾木」が現実的。
  - 床材が切り替わる場所（フローリング⇔CF等）には床見切り材。敷居・框との段差処理も見切りで解決。
  - 無垢は壁と擦れて床鳴りすることがある→巾木を外しクリアランス確保で解消。
- **情報源**: https://www.yukazaiya.com/hpgen/HPB/entries/16.html / https://diy-ch.com/kisochisiki/3058/ / https://report-takahashi.com/2021/12/23/納まりについて/

### 2-6. 和室からの改装（畳→フローリング）
- **読者の悩み**: 「畳を剥がしたら下はどうなってる？」「高さ60mmをどう埋める？」
- **押さえるべき事実**:
  - 畳厚は約55〜60mm。定番構成は根太+構造用合板12mm+フローリング15mm（+調整ベニヤ）。
  - 根太間隔は450mm（頑丈にするなら300mm）。本数=(部屋幅mm÷303)+1。下地は1〜5mmの不陸があるので2〜5mm厚ベニヤで水平調整。
  - 代替: スタイロフォーム+合板で高さ調整する方法（マンション和室で人気、断熱も取れる）。
  - 30mm厚フローリングなら根太+直張りで合板省略可。
- **情報源**: https://99diy.tokyo/washitu_tatami-floor/ / https://99diy.tokyo/japaneseroomfloor_foundation/ / https://morinogakko.jp/apartment-muku/ / https://www.diy-shop.jp/second/flooring/tatami.html

### 2-7. マンションの防音規定
- **読者の悩み**: 「管理規約のLL45って無垢じゃ無理？」
- **押さえるべき事実**:
  - 多くの管理規約がLL45（ΔLL(I)-4）以上を要求。無垢15mm単体では不可。
  - クリア方法: ①遮音マット11mm+合板12mm+無垢15mm（計38mm）②遮音二重床（置き床）の上に施工 ③専用遮音マット（サウンドプルーフ等22mm）併用 ④遮音等級付き挽板複合に切替。
  - 二重床マンションでは仕上げ材にLL45が本来不要という専門家の問題提起もある（管理組合との交渉論点）。
  - 工事には管理組合への申請・承認が必要。
- **情報源**: https://www.zenkoku-mankan.org/floor-grade/ / https://www.muku-flooring.jp/l45_flooring/ / https://www.muku-flooring.jp/blog/2025/10/31/56114/ / https://www.eco-inc.co.jp/qa/2017/03/l40.php

### 2-8. 失敗例とリカバリ・日数・難易度
- **読者の悩み**: 「素人でも本当にできる？」「何日かかる？」
- **押さえるべき事実**:
  - 工期目安: 業者の張替えで6畳2〜3日。DIY上張りで6畳約10時間、経験者2人の30mm根太直張りで6時間、初心者の畳→フローリングは週末2〜3回が現実的。
  - 頻出失敗: 部屋の中央から張って半端材が目立つ/まっすぐ切れず隙間/材料不足（ロス率5〜10%を見込まず発注）/水性ボンド使用で床鳴り/スペーサー抜き忘れ/クリアランスゼロで突き上げ/ドア干渉。
  - 「フローリング張りは誰もが難関と書く」が、上張り・敷くだけ式なら初心者可。剥がし張替えと下地調整は難易度高。
- **情報源**: https://satoyamakurasi.com/flooring-diy/ / https://ameblo.jp/tpms0419/entry-12617528678.html / https://2103104.net/745.html / https://shimotori.jp/i-repaper-it-and-market-price-days-and-diy-of-the-expense-comment-on-whether-they-can-do-it-exchange-of-the-flooring/

### 2-9. 必要枚数の計算
- **読者の悩み**: 「6畳で何ケース買えばいい？」
- **押さえるべき事実**:
  - 1坪=3.24㎡(物販慣習)、1ケース=半坪（約1.66㎡）が標準（90mm幅×1820mmで10枚、75mm幅12枚、150mm幅6枚）。
  - 必要量=施工面積×1.05〜1.1（ロス率）。乱尺はロス多め。自動計算ツールを置く店が多い（RESTA、マルキョク等）＝**自社サイトに計算ツール設置はSEO/CV両面で有効**。
- **情報源**: https://www.diy-shop.jp/info/autosize_flr.html / https://www.prairie.co.jp/blog/?p=78 / https://www.marukyoku.com/flooring/calculation/ / https://hayazai.com/blog/how-many-sheets.html（既存記事）

## 領域3: お手入れ・メンテナンス

### 3-1. 日常の掃除（掃除機・水拭き・ロボット掃除機）
- **読者の悩み**: 「水拭きしていい？」「ルンバ・ブラーバは使える？」
- **押さえるべき事実**:
  - 基本は掃除機 or 箒+乾拭き。水拭きは「固く絞れば可」だが毎日は不要・非推奨。無塗装/オイルは水分残りで毛羽立ち・シミに。
  - 吸引型ロボット掃除機は基本OK（砂粒の噛み込みに注意）。**水拭きロボット掃除機はNG**（水量調整ができず反り・カビの恐れ。オイル仕上げは特に不可）。
  - 化学モップ・市販の樹脂ワックス（リンレイ等）は無垢にNGの場合が多い（白濁・密着不良）。
- **情報源**: https://www.cas-online.jp/blog/feature/8726/ / https://www.muku-flooring.jp/blog/2020/05/10/23309/ / https://www.mokuzai.com/Faq/264 / https://ecofield.jp/blog/17305.html / https://miniit.net/reforming/naturalmaterials/wax/

### 3-2. 無塗装品の手入れ
- **読者の悩み**: 「無塗装のまま使っていい？」「黒ずんできた」。
- **押さえるべき事実**:
  - 箒掃き+固く絞った雑巾がけが基本。汚れ・毛羽立ちはサンドペーパー（240番程度）で研磨できるのが無垢の強み。
  - 手垢黒ずみ・皮脂汚れは中性洗剤→アルカリが強い洗剤は変色するので注意。
  - 無塗装でも後からオイル・蜜蝋を塗って「育てる」移行が可能。
- **情報源**: https://www.muku-flooring.jp/blog/2021/03/15/44242/ / https://daityu.shop/column/clean.html / https://www.atopico.com/blog/52993.html

### 3-3. オイル・ワックスの再塗装（蜜蝋・オスモ・キヌカ等）
- **読者の悩み**: 「何を塗ればいい？」「頻度は？」「商品が多すぎて選べない」。
- **押さえるべき事実**:
  - 頻度目安: キッチン・トイレ等ハードな場所は半年〜1年、リビング1〜2年、寝室2〜3年。
  - 製品比較: キヌカ（米油系・無臭・自然発火しにくい・乾燥24時間以上・防水力は控えめ）、オスモ（塗膜感あり・防汚強い）、蜜蝋ワックス（ナチュラル・塗りやすいが塗膜弱くメンテ頻度高め）、AURO等。
  - 「初回はオイル、メンテは蜜蝋」という併用パターンが実用的。塗布は薄く伸ばす・ウエスの自然発火に注意（油系は水に浸して廃棄）。
- **情報源**: https://www.chachasora.com/comparison-7types-solidflooringwax/ / https://www.tanakaseizai.jp/comparison-of-paint-coating/ / https://www.muku-flooring.jp/maintenance/oil.html / https://www.livlan.com/temahima/magazine/13501

### 3-4. ウレタン塗装品の手入れ
- **読者の悩み**: 「ウレタンなら何もしなくていい？」
- **押さえるべき事実**: 日常掃除のみでOK・ワックス不要。ただし塗膜が割れた傷は自分で部分補修不可（再塗装は業者・全面研磨）。年間コストはオイルより安いがリカバリ性で劣る。
- **情報源**: https://daityu.shop/column/urethane.html / https://mukurich.jp/muku-flooring-maintenance-oil-urethane/

### 3-5. 加湿器・結露・湿度管理
- **読者の悩み**: 「加湿器を置くと床が傷む？」「最適湿度は？」
- **押さえるべき事実**:
  - 最適湿度40〜50%前後で寸法が安定（平衡含水率）。過乾燥は隙間・割れ、過湿は膨張・突き上げ。
  - 加湿器の蒸気吹き出し口直下・水タンクの結露水たまりは輪じみ/カビの原因→台やマットを敷く。
  - 観葉植物の鉢・濡れバスマット・布団敷きっぱなしはカビの3大原因。
- **情報源**: https://www.replan.ne.jp/articles/3296/ / https://www.cas-online.jp/blog/feature/7634/ / https://www.muku-flooring.jp/blog/2022/09/25/51276/

## 領域4: トラブル対処

### 4-1. 凹み・傷の補修（アイロン法）
- **読者の悩み**: 「物を落として凹んだ。直せる？」
- **押さえるべき事実**:
  - 手順: ①凹みにマチ針で小穴 ②霧吹き or 濡れタオルで水分 ③濡れ布越しにスチームアイロン数回 ④乾燥→毛羽立ちはペーパー研磨 ⑤オイル/ワックス再塗布。
  - 適用は無塗装・オイル仕上げのみ。ウレタン/UV塗装は塗膜が白濁・剥離するので不可（業者リペア）。
  - 早いほど戻りやすい。繊維が切れた「えぐれ傷」は完全には戻らない。浅い傷はサンドペーパーで消える。
- **情報源**: https://www.mokuzai.com/contents/blog/2022-0504/ / https://ecoloquia.co.jp/20221104-01/ / https://www.livlan.com/temahima/magazine/13416 / https://www.eco-inc.co.jp/natural-material/2007/01/post_114.php

### 4-2. 隙間・反り・突き上げ
- **読者の悩み**: 「冬に隙間ができた。施工不良？」「床が盛り上がってきた」。
- **押さえるべき事実**:
  - 原因は調湿による膨張収縮。幅方向は長さ方向の約10倍動く。冬収縮・夏膨張のサイクル。
  - 施工時クリアランス（板間0.2〜0.5mm・壁際5〜10mm）不足、床下換気不足、水性ボンド使用が突き上げの典型原因。
  - 「季節変動の隙間は1年様子を見る」が専門家の定番回答。隙間に詰まったゴミはハケ・爪楊枝・隙間ノズルで。
  - 湿度50%前後の管理が最大の予防。
- **情報源**: https://www.atopico.com/mukuzai/blog/25950.html / https://www.nishibayashi-kk.co.jp/catalog/notice.html / https://www.cas-online.jp/blog/feature/7634/ / https://sumika.me/seek_advices/2219

### 4-3. 水濡れ・シミ・輪じみ
- **読者の悩み**: 「水をこぼして放置したらシミに」「コップの輪じみが取れない」。
- **押さえるべき事実**:
  - 直後対応: 乾いた布で押さえて吸水→二度拭き→換気乾燥。
  - 残ったシミ: ウレタンは固絞り雑巾で拭くだけ。オイル/無塗装はスポンジ硬面 or サンドペーパーで表面を削り→再オイル。
  - 黒ずみ系のシミ（金属+水分のタンニン反応）はお酢など酸で中和すると薄くなる場合あり。
  - 塩素系漂白剤は絶対NG（変色・劣化）。
- **情報源**: https://www.muku-flooring.jp/blog/2020/05/05/23186/ / https://www.atopico.com/blog/52993.html / https://daityu.shop/column/clean.html / https://tennenmokuzai.com/column/290/

### 4-4. カビ
- **読者の悩み**: 「布団の下にカビが生えた」。
- **押さえるべき事実**:
  - 原因: 濡れバスマット放置・鉢直置き・布団敷きっぱなし・換気不足。
  - 軽度: 中性洗剤を吹いて3〜5分→固絞り雑巾→アルコールスプレー→乾拭き。重度: サンディングで削り再塗装。
  - 塩素系漂白剤NG。予防は換気+乾拭き+寝具の除湿+定期オイル。桧は抗菌成分でカビに比較的強い（訴求点）。
- **情報源**: https://www.muku-flooring.jp/blog/2022/09/25/51276/ / https://hags-ec.com/column/floor-210547-m/ / https://uchi.tokyo-gas.co.jp/topics/6001

### 4-5. 日焼け・経年変化
- **読者の悩み**: 「色が変わってきたけど大丈夫？」「ラグの跡だけ白い」。
- **押さえるべき事実**:
  - 桧は白い木肌→黄〜赤みの飴色へ。紫外線+油分の変化による自然現象で「劣化ではなく成長」。摩擦と皮脂で艶が増す。
  - ラグ・家具の下だけ色が変わらず跡になる→定期的に位置を変える/一定期間で日に当てると追いつく。
  - 樹種ごとの経年比較写真はコンテンツとして強い（キャスオンラインが先行）。
- **情報源**: https://ecoloquia.co.jp/20251208-01/ / https://andonly.co.jp/keinen/ / https://www.muku-flooring.com/faq/about-muku/3906/ / https://www.cas-online.jp/blog/pickup/6586/

### 4-6. きしみ・床鳴り
- **読者の悩み**: 「歩くとミシミシ鳴る。家が傷んでる？」
- **押さえるべき事実**:
  - 音で原因診断: ギシギシ=板の継ぎ目の擦れ（季節性）、キュッキュ=釘の緩み、ミシミシ=膨張で板同士が圧迫。
  - 無垢特有: 冬の収縮/夏の膨張による摩擦音は欠陥ではない。膨張圧迫はサネ目地に切れ込みで逃げを作ると止まることがある。壁との擦れは巾木を外してクリアランス確保。
  - 市販の床鳴り止め液剤は「100%無垢には使えない」品があるので注意。下地由来は床下補強が必要＝業者案件の見極え。
- **情報源**: https://reform.cainz.com/knowledge/common/8558 / https://www.muku-flooring.com/faq/about-maintenance/2461/ / https://www.nikka-home.co.jp/column/hint/20250922142117.html / https://lifeup-floor.com/column/170/

### 4-7. 水回り（洗面・脱衣所）での使用
- **読者の悩み**: 「脱衣所も桧にしたいが腐らない？」
- **押さえるべき事実**:
  - 桧は油分・抗菌成分で針葉樹中もっとも水に強い部類（桧風呂・温泉脱衣場の実績）。
  - ただし「毎回完全に拭き取る」運用は現実には難しく、シミを許容できる人向け。風呂上がりの素足に桧は最高、という体験訴求とセットで。
- **情報源**: https://www.honest-p.com/blog/interior-wood/flooring/p8945/ / https://nasahome.co.jp/magazine/dressingroom_floor/ / https://www.ookubo-k.com/blog/865.html

### 4-8. 浮造り（うづくり）仕上げ
- **読者の悩み**: 「浮造りって何？滑りにくいって本当？」
- **押さえるべき事実**: 年輪の柔らかい部分を磨き落として凹凸を出す加工。滑りにくい（ペット・高齢者・濡れ足）、空気層で温かい、足裏刺激、傷が目立ちにくい。反面、加工コスト高・凹凸に汚れが入りやすい。
- **情報源**: https://www.woodone.co.jp/media/cat08/3528/ / https://kashida-m.co.jp/column/202506_03/ / https://niigata.jutaku2shin.com/column/22919/

## 「読者が本当に知りたいのに多くのサイトが答えていない『かゆいところ』」

1. **桧に特化した「後悔・デメリット」記事が少ない**。「無垢フローリング 後悔」は激戦だが「桧フローリング 後悔/デメリット」を工場目線で正直に書いた記事は薄い。製造者が自分でデメリットを語ると信頼性が突き抜ける。
2. **「ヒノキチオールは国産桧にほぼ含まれない」という正確な情報**。ほぼ全サイトが曖昧にぼかしている。正しい抗菌成分（α-カジノール等）を語れるのは材木店ならでは。
3. **1ケース・1坪・1畳・㎡の換算と「ロス率込みの発注数」の実践解説**。計算ツール+乱尺/UNI別のロス率目安まで踏み込んだ記事はほぼない（既存記事の強化ポイント）。
4. **最後の一列・敷居まわり・出隅入隅など「DIYの詰みポイント」だけを集めた記事**。手順記事は多いが「詰んだときのリカバリ集」は希少。
5. **水性ボンドNG・サネにボンNG・スペーサー抜き忘れなど「やってはいけない集」**。失敗談ブログには散在するが、専門店がまとめた決定版がない。
6. **床暖房対応品と非対応品を同室併用してはいけない**という注意。ほぼ材木屋1社しか明言していない。
7. **水拭きロボット掃除機（ブラーバ等）の可否**。需要急増中なのに専門店の回答が少なく、メーカーFAQ頼み。
8. **「隙間は1年待て」という季節変動の考え方**。クレームになる前に読ませたい予防線コンテンツ。施工不良との見分け方まで書くと唯一無二。
9. **サンプルの正しい見方**（照明で色が変わる・小片と全面の印象差・天然のばらつき）。通販直販店こそ書くべきで、CVの不安解消に直結。
10. **脱衣所・洗面所に桧は使えるか**の本音回答。「桧は水に強いがシミ許容の覚悟次第」という二段構えで答える記事は少ない。
