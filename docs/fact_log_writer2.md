## 計算ツールの手計算検証（3ケース）

ロジック: 床面積（畳数×1.62 or ㎡）× 形状補正 × ロス率 ÷ 1ケース㎡ → 切り上げ。
1ケース㎡はdata/products.jsのflooring15実規格から動的取得（910mm:20枚=1.9656㎡ / 1820mm:20枚=3.9312㎡ / 3000mm:10枚=3.24㎡ / 4000mm:8枚=3.456㎡）。

| ケース | 入力 | 手計算 | ツール出力（node検証） | 一致 |
|---|---|---|---|---|
| A | 6畳・1820mm品・補正なし・ロス10% | 6×1.62=9.72㎡ → ×1.10=10.692㎡ → ÷3.9312=2.7198 → **3ケース（60枚・11.79㎡）** | 3ケース/60枚/11.7936㎡ | ✅ |
| B | 8畳・910mm品・補正+5%・ロス5% | 8×1.62=12.96 → ×1.05×1.05=14.2884 → ÷1.9656=7.2692 → **8ケース（160枚・15.72㎡）** | 8ケース/160枚/15.7248㎡ | ✅ |
| C | 12㎡・3000mm品・補正なし・ロス10% | 12×1.10=13.2 → ÷3.24=4.0741 → **5ケース（50枚・16.2㎡）** | 5ケース/50枚/16.2㎡ | ✅ |

記事内の計算例（6畳→3ケース）はケースAと同一で整合。インラインJSは構文チェック済み（new Functionでパース成功）、PRODUCTS未定義時のフォールバック規格も同値で実装。

## 裏が取れず表現を弱めた・削除した論点まとめ

1. OPC製造量10%以下 → 1ソースのみ。「1割以下ともいわれます」に弱めた。
2. 乱尺の長さ範囲 → ソース間で300mm以上/400〜1200mmと差。「おおむね300〜1,200mm程度」と幅を持たせた。
3. 乱尺のロス・施工手間 → 専門店間で見解が割れるため両論併記。
4. UNIと乱尺の価格上下 → ソースで逆転するため順位付けせず。
5. 30mm根太直張り → 「可能とされています」+ 捨て貼り推奨派の存在と施工者確認の注意書きを必ず添えた。
6. 30mmの断熱効果 → 「期待できるとされています」+ 床下断熱材の方が効果的という専門店見解を併記。
7. ウレタン塗膜寿命20〜30年 → 2ソース確認できず数値を記事から削除。
8. グレードと強度 → 「無節の方が強い」とする製材所記述があったため、「床材としての実用上の差はないと考えて差し支えない」と用途限定の緩和表現に。
9. 幅別入数の慣習値（75mm12枚・150mm6枚） → 2ソース確認できず記事に不採用。確認できた90mm10枚・120mm7枚のみ記載。
10. 1畳=1.62㎡は厳密には不動産表示基準（中京間3尺×6尺は約1.65㎡）→「中京間にほぼ相当する基準値」と表現し、地域差と実測推奨を明記。
