## 4. 残存リスク（軽微・対応不要と判断したもの）

1. **「2008年のJIS改正」（mansion-ll45）**: 厳密にはJIS A 1440制定は2007年・ΔL等級表示制度の運用開始が2008年とされるが、引用元（マンション管理組合連合会サイト）自体が「2008年のJIS規格改正」と表記しており、出典準拠のため非修正。
2. **再塗装頻度の出典割れ**: 「年1〜2回」説（ムクリッチ等）も実在するため、今回統一した「場所別・半年〜2、3年」が唯一の正解ではない。ただし全記事で同じ目安に揃えており読者が混乱するリスクは解消。
3. **キヌカの乾燥時間**: 調査ファイル（blog_research_2026-06-12.md 3-3）には「乾燥24時間以上」と記載が残っている。記事側は正しいが、調査ファイルを再利用する将来の執筆者が誤記を引き継ぐ恐れあり（記事は修正済みのため実害なし）。
4. **12mm規格の方針**: 「12mmは15mmに統合」という過去決定が再び有効化された場合、kasanebari-diy / thickness-15-30 / how-many-sheets / flooring-vs-panel / uni-opc-ranjaku / kugi-screw-bond の12mm言及（計6記事）を一括で見直す必要がある。現時点では products.js に12mmが実在するため全記事正当。
5. **JSON-LDのmainEntityOfPage / image プロパティ**: 一部記事で未設定だが、Articleスキーマの必須項目ではなくSEO上の軽微な改善余地に留まる。
6. **blog.html / sitemap.xml**: 指示どおり未確認・未編集（統合担当の管轄）。
